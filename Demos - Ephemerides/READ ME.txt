-- READ ME --

This folder contains:

- 'MAIN_test_EPH.m' file is used to compare ephemerides generation for major
  planets of Solar System with three different methods (polynomials
  evaluations, approximate keplerian elements and circular-coplanar model)
- 'MAIN_test_EPH_2.m' file is used to compare circular-coplanar and Horizon
  velocities for Mars
- '.txt' files for Horizon ephemerides of major planets in June 2003 -
  September 2015 timeframe
- MATLAB '.m' functions needed to run 'MAIN_test_EPH.m' without getting any
  error

Enjoy! :D
